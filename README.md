# Harvey Blocks' Space

A clean and modern personal space for hosting and rendering markdown-based content directly from GitHub.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Features

- **GitHub Pages Ready**: Optimized for multi-project hosting on GitHub Pages.
- **Markdown Rendering**: High-quality markdown rendering with support for GFM, Math (KaTeX), and Syntax Highlighting.
- **Dynamic File Tree**: Automatically fetches and displays file structure from your content repository.
- **Responsive Design**: Built with Tailwind CSS for a seamless experience across all devices.
